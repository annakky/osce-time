# 의대생 시험 진행 타이머

정적 웹앱(HTML/CSS/JS)이며 GitHub Pages에 바로 배포할 수 있습니다.

## 기능
- 타이머 시작/멈춤/초기화
- 기본 12분, 시작 전 시간(분) 설정 가능
- 랩 기록 기능
- 음성 안내: `시험시작` / `종료 2분전` / `시험 종료`
- 0초 이후에도 초과 시간이 계속 증가(+MM:SS)
- 체크리스트 선택 후 시험 진행
- 체크리스트 수정 팝업(추가/삭제/이름변경/항목 추가/삭제)
- 수정 팝업에서 `완료`를 눌렀을 때만 반영
- 수정 팝업에서 항목 드래그로 순서 변경(끼워넣기 형태, 모바일 터치 지원)
- 체크리스트 초기화 버튼(로컬스토리지 삭제 후 txt 원본 재로딩)
- 로컬스토리지 저장(DB 대체)
- `txt` 파일 기반 체크리스트 초기 로딩

## 체크리스트 txt 규칙
- 파일명(`.txt` 제외)이 체크리스트 이름
- 파일의 줄 단위 원문 그대로 체크 항목으로 사용
- `1.` 같은 번호 텍스트를 제거하지 않음
- 한 파일 안에서 번호가 다시 `1.`로 시작해도 분리하지 않고 같은 체크리스트로 유지
- 로딩 대상 파일 목록은 `checklists-manifest.json`에서 관리

## 로컬스토리지 동작
- 첫 접속(로컬스토리지 비어 있음): txt에서 체크리스트를 불러와 저장
- 이후 접속: 로컬스토리지 데이터 우선(사용자 수정 내용 유지)
- 로컬스토리지 삭제 후 재접속: txt 원본으로 다시 초기 로딩

## 실행
`txt` 자동 로딩을 위해 로컬에서도 HTTP 서버로 실행하세요.

예시(PowerShell):
```powershell
python -m http.server 5500
```
브라우저에서 `http://localhost:5500` 접속

## 배포 (GitHub Pages)
1. 이 폴더를 GitHub 저장소에 푸시
2. 저장소 `Settings > Pages`에서 배포 브랜치 설정 (예: `main` / root)
3. 배포 URL 접속

## 저장소 데이터
- 키: `med-exam-timer-v1`
- 저장값: 시험 시간 설정, 체크리스트 목록, 선택된 체크리스트
