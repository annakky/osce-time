# 의대생 시험 진행 타이머

정적 웹앱(HTML/CSS/JS)이며 GitHub Pages에 바로 배포할 수 있습니다.

## 기능
- 타이머 시작/멈춤/초기화
- 기본 12분, 시작 전 시간(분) 설정 가능
- 랩 기록 기능
- 음성 안내: `시험시작` / `종료 2분전` / `시험 종료`
- 0초 이후에도 초과 시간이 계속 증가(+MM:SS)
- 체크리스트 선택 후 시험 진행
- 체크리스트 추가/삭제/이름변경/항목 추가/삭제
- 로컬스토리지 저장(DB 대체)

## 실행
로컬에서 `index.html`을 브라우저로 열면 됩니다.

## 배포 (GitHub Pages)
1. 이 폴더를 GitHub 저장소에 푸시
2. 저장소 `Settings > Pages`에서 배포 브랜치 설정 (예: `main` / root)
3. 배포 URL 접속

## 저장소 데이터
- 키: `med-exam-timer-v1`
- 저장값: 시험 시간 설정, 체크리스트 목록, 선택된 체크리스트
